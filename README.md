  #### 🧱 Featured project

  )** — a production-grade React monorepo with enforced
  layer architecture, multi-platform support (web + admin +
  Tauri desktop), design-system package, generators, typed
  command/event bus, MSW mock API, and batteries-included

  #### 💡 What we build around

  - **Opinionated by default** — no bikeshedding over folder
   layout
  - **Architecture enforced by tooling** — ESLint
  boundaries, not just convention
  - **Cross-platform code sharing** — web, admin, desktop
  from one codebase
  - **Developer ergonomics** — generators, mock APIs, design
   systems, batteries included
  - **Security-first** — automated audits, license policies,
   dependency hygiene
